# VistaReyclerView
一个提供`加载更多`和`下拉刷新`的清真派RecyclerView

## Import

Maven

	<dependency>
  <groupId>me.shaohui.VistaRecyclerView</groupId>
  <artifactId>app</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
    
Gradle

	dependencies {
        compile 'me.shaohui.VistaRecyclerView:app:1.0.0'
     }
